# Xmas Vacation Website Specification

A beautiful, interactive winter-themed website celebrating the holiday season and New Year's countdown.

## Core Requirements

### 1. Landing Page
- Captivating description of the Christmas vacation experience.
- Hero section with a festive winter theme.

### 2. Highlight of the Year
- A dedicated section showcasing the most memorable moments of 2025.

### 3. Design & Styling
- **Framework**: React.
- **Styling**: Vanilla Extract CSS (Type-safe, zero-runtime CSS-in-TypeScript).
- **Theme**: Beautiful winter theme (whites, blues, festive accents, snowflake patterns).

### 4. Responsive Design
- Fully mobile-responsive layout.
- Optimized for desktop, tablet, and mobile screens.

### 5. Deployment
- Hosted on **Vercel**.
- Automated deployment with a shareable live URL.

### 6. Photo Gallery
- Grid or Carousel layout.
- Using high-quality free images (Unsplash/Pexels).

### 7. Countdown Timer
- Real-time countdown to New Year 2026.

### 8. Testimonials
- A page or section featuring vacation stories/testimonials from family/friends.

### 9. Interactive Elements
- CSS/Framer Motion animations.
- Hover effects on cards and buttons.
- Possible snowflake falling effect.

## Technical Stack
- **Frontend**: Next.js (React)
- **Styling**: @vanilla-extract/css
- **Deployment**: Vercel
- **Animations**: Framer Motion or Vanilla CSS transitions
